Interactive Visualization

Upload a text file of SMILES strings and this app will create a plot that visualizes:

  • how easy it is to synthesize each compound in that file (y-axis). This synthetic accesibility score is calculated according to work from Ertl and Schuffenhauer
  • How lipophilic (clogP) each compound is (x-axis). This is calculated according to work from Wildman and Crippen
  • Display the molecular structure upon hoovering

The file must be:

  • .txt
  • each line in this file must contain exactly one valid SMILES string
  • smaller than 5 KB
